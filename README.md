# Contemporary Education

This is a MkDocs site covering various aspects of contemporary education in India.

## Units Covered

1. Education in Contemporary India
2. Understanding Social Diversity
3. Educational Demands of Individuals and Diverse Communities
4. Language Policy in Education
5. Implications of Equality of Educational Opportunities

## Local Development

To run the site locally:

```bash
mkdocs serve
```

Then visit http://localhost:8000

## Deployment to GitHub Pages

This site is configured for deployment to GitHub Pages.

### Steps to Deploy:

1. Create a new repository on GitHub named `contemporary_education`
2. Add the remote repository:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/contemporary_education.git
   ```
3. Push the main branch:
   ```bash
   git push -u origin main
   ```
4. Deploy to GitHub Pages:
   ```bash
   mkdocs gh-deploy
   ```

The site will be available at: `https://YOUR_USERNAME.github.io/contemporary_education/`
